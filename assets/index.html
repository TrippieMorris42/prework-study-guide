HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language.
  • The structure elements of a webpage include;
  • Text elements, Image elements, Header elements, List elements

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • Declarations means to contain two important components;
  • CSS property we want to apply and the value of the property.
  • CSS stands for Cascading Style Sheets
  • CSS is used for specifying the presentation and styling of a webpage
